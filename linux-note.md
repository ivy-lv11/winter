# "HH's Guide to Lunix"  
ctrl shift N   open a new terminal  
ctrl ++ zoom 
ctrl + - zoom out

## in tmux tmux使用 
ctrl B % 
ctrl B "
ctrl B x quit  
ctrl B j/l/k/h

## switch between the multitasks 多任务切换 
Alt + Tab  

## switch between the desktop 切换桌面 
ctrl  Alt left/right/up/down  

## less /etc/passwd  
display the info in a screen. use "space" to move forward  
and use "b" back  
"q" to quit.
"/" to search  
"^" at the beginning  

## cal日历使用
display a calendar for a particular year.  
eg: cal 1952  
cal 2019 | less  
cal 12 1952		//specify a month  

## whoami  
whoami //lv
sudo whoami //root  

## Tab补全
不仅仅是文件名，还有命令也可以  
例如whoa<Tab> 得到 whoami  
 
## 查看磁盘使用和剩余空间  
df
