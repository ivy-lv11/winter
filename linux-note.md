# "HH's Guide to Lunix"  
ctrl shift N   open a new terminal  
ctrl ++ zoom 
ctrl + - zoom out

## in tmux  
ctrl B % 
ctrl B "
ctrl B x quit  
ctrl B j/l/k/h

## switch between the multitasks  
Alt + Tab  

## switch between the desktop  
ctrl  Alt left/right/up/down  

## less /etc/passwd  
display the info in a screen. use "space" to move forward  
and use "b" back  
"q" to quit.
"/" to search  
"^" at the beginning  

## cal
display a calendar for a particular year.  
eg: cal 1952  
cal 2019 | less  
cal 12 1952		//specify a month  

## whoami  
whoami //lv
sudo whoami //root
